# 🤖 AI Voice Sales Agent (N8N + AmoCRM + Vapi)

**Интеллектуальный голосовой помощник для стоматологии.**

Этот проект демонстрирует связку современных AI аудио-технологий (Vapi, ElevenLabs) и автоматизации n8n с популярной CRM-системой (AmoCRM). Система самостоятельно звонит клиентам (или принимает звонки), общается голосом, собирает первичный анамнез, проверяет свободных врачей и заносит сделку в CRM.

## 📊 Бизнес-ценность
* **Захват 100% лидов**: Мгновенный ответ за 15 секунд.
* **Повышение конверсии**: +35% за счет быстрого контакта.
* **Экономия ресурсов**: Заменяет 2 полноценных операторов call-центра на первой линии.

---

## ⚙️ Архитектура системы

Логика бота построена в **n8n**. Workflow обрабатывает входящие Webhooks от Vapi, делает API-запросы к AmoCRM и возвращает клиенту контекстные ответы (например, доступное время для записи).

### Основные компоненты:
1. **n8n (Оркестрация)**: `Dental AI Agent → AmoCRM.json` - главный обработчик.
2. **Анализ расписания**: `check_availability_workflow.json` - модуль проверки времени врачей.
3. **Голосовой канал**: Vapi (Speech-to-Text & Text-to-Speech) + ElevenLabs для ультра-реалистичного звука.
4. **CRM**: AmoCRM API.

## 🛠 Технологический стек
