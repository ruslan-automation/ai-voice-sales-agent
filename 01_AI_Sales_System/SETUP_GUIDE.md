# 🛠️ Настройка доступов для "AI-РОП" (AI Sales Head)

Чтобы наша система заработала, нам нужно получить ключи от двух сервисов: **AmoCRM** и **ElevenLabs**.

---

## 1. AmoCRM (14 дней Trial)
1.  Зайди на [amocrm.ru](https://www.amocrm.ru/) и нажми **"Попробовать бесплатно"**.
2.  Зарегистрируйся (почта + телефон).
3.  В аккаунте перейди в **Настройки (шестеренка)** -> **Интеграции**.
4.  Нажми кнопку **"Создать интеграцию"** (вверху справа).
5.  Выбери **"Внешняя интеграция"**.
    *   **Ссылка для перенаправления**: `https://oauth.n8n.io/v2/oauth2-callback` (Это стандартный callback для n8n cloud, если у тебя локальный - используй свой).
    *   **Предоставить доступ**: Выбери "Все" (или CRM, Сделки, Контакты).
    *   **Название**: `n8n Portfolio`
    *   **Описание**: `Test`
6.  Сохрани.
7.  Открой созданную интеграцию, скопируй:
    *   `ID интеграции` (Client ID)
    *   `Секретный ключ` (Client Secret)
    *   `Код авторизации` (Он действует 20 минут, понадобится сразу при подключении в n8n).

---

## 2. ElevenLabs (Voice AI)
1.  Зайди на [elevenlabs.io](https://elevenlabs.io/).
2.  Sign Up (можно Google аккаунт).
3.  Нажми на иконку профиля (справа вверху) -> **Profile + API Key**.
4.  Нажми **"View"** напротив API Key и скопируй его.
    *   `API Key`: `sk_...`
5.  Перейди в раздел **"Conversational AI"** (в меню слева).
6.  Нажми **"Create Agent"**.
    *   Выбери шаблон "Blank" или "Sales".
    *   Выбери голос (Voice).
    *   В настройках агента найди **Agent ID** (нужен будет для n8n).

---

## 3. Куда это вставлять?
Мы откроем n8n и создадим "Credentials".
*   Для **AmoCRM**: Тип `AmoCRM OAuth2 API`.
*   Для **ElevenLabs**: Тип `Header Auth` (Name: `xi-api-key`, Value: `твои_ключ`).
